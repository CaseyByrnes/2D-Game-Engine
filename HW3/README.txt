For Homework 3, each part is in it's own subfolder. The keyboard controls as well as the instructions for compiling and running each part are listed below.

KEYBOARD CONTROLS:
Move left: "A"
Move right: "D"
Jump: "Spacebar"
Sprint: "Left Shift"
Pause/Unpause: "P"

Adjust timing: "T" +
    0.5 Time: "1"
    1.0 Time: "2"
    2.0 Time: "3"


*********** PART 1 ***********
Navigate to the "Part_1" folder and compile using the Makefile included with the project in the HW3 folder. (The Makefile is the example Makefile given to the class with some modifications.)

Once compiled, run using "./main" in terminal.
